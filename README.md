#  NSUserActivityDelete

This project is intended to show that donated Siri Shortcuts aren't being deleted when requested to do so
using the NSUserActivity API.

1.    Build and Launch NSUserActivityDeleteTest application on iOS 13 or greater
2.    Press the Create Activity button
3.    Return to the Home Screen.  Pull down to enter Search.
4.    Verify that the Shortcut was donated by searching for "NSUserActivity" 
5.    Verify that you see a Shortcut for the application "NSUSERACTIVITYDELETETEST"
6.    Return to the NSUserActivityDeleteTest application
7.    Press the Delete Activity button
8.    Return to the Home Screen and perform the same steps to find the donataed Shortcut
9.    Invalid Result:  The donated shortcut still appears
